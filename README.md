# MainChainGaugeInjector

## Usage

### Build

```
$ forge build
```

### Test

```
$ forge test
```

### Format

```
$ forge fmt
```

### Gas Snapshots

```
$ forge snapshot
```

### Deploy

```
$ forge script --chain mainnet script/Deploy.s.sol:Deploy --rpc-url $ETHEREUM_RPC_URL --broadcast --verify -vvvv
```
